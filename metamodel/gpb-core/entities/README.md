## Файловая структура папки entities

```

|- app                      - Прикладная архитектура
|  |- aspects               - Расширение для дефолтной сущности aspects
|  |- components            - Расширение для дефолтной сущности components
|  |- contexts              - Расширение для дефолтной сущности contexts
|  |- documents             - Расширение для дефолтной сущности documents
|  |- libraries             - Кастомная сущность для управления библиотеками
|  |- sequence_diagrams     - Пример от Романа Пионтика по управлению диаграммами последовательности, пока не подключена
|  |- root.yaml             - Импорт сущностей и расширений
|- ba                       - Бизнес-архитектура
|  |- org_structure         - Описание кастомной сущности для организационных струтур ГПБ (департамент, VS, команда VS)
|  |- subunits              - Описание кастомной сущности подразделения (обобщенная модель)
|  |- root.yaml             - Импорт сущностей и расширений
|- dochub_menu              - Кастомная сущность для управления структурой меню DocHub
|  |- templates             - Шаблон дерева меню (PlantUML)
|  |- dochub_menu_model.yaml- Описание кастомной сущности для управления структурой меню DocHub
|  |- root.yaml             - Импорт сущностей и расширений
|- ia                       - Информационная архитектура
|  |- business_entities     - Кастомная сущность для управления НСИ и бизнес-сущностями на логическом уровне
|  |- dictionaries          - Кастомная сущность для управления справочной информацией
|  |- root.yaml             - Импорт сущностей и расширений
|- ta                       - Техническая архитектура 
|  |- environments          - Кастомная сущность для управления средами развертывания систем
|  |- root.yaml             - Импорт сущностей и расширений

```
